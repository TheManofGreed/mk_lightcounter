# Плаваюча таблиця підрахунку годин для off.energy.mk.ua (Tampermonkey)

**Опис:**  
Цей користувацький скрипт для Tampermonkey підраховує кількість годин, коли є світло, немає світла та можливо світло (Є / Немає / Можливо) у таблицях з класом `.tabSchedule`. Скрипт створює окрему плаваючу таблицю у правому верхньому куті сторінки, щоб ви могли одразу бачити результати, не змінюючи оригінальну таблицю.

**Функції:**
- Підраховує "Є" (світло увімкнено), "Немає" (світло вимкнено) та "Можливо" (можливе світло) для всіх позицій.
- Відображає результати у плаваючій таблиці з кольоровою маркуванням:
  - Зелений — Є
  - Червоний — Немає
  - Помаранчевий — Можливо

**Встановлення:**
1. Встановіть [Tampermonkey](https://www.tampermonkey.net/) у ваш браузер.
2. Завантажте файл [`mk_LightCounter.user.js`](./mk_LightCounter.user.js) або скористайтеся посиланням на raw GitHub.
3. Відкрийте файл — Tampermonkey запропонує встановити скрипт.

**Використання:**
- Перейдіть на сторінку з таблицею `.tabSchedule`.
- Скрипт автоматично створить плаваючу таблицю у правому верхньому куті.
